NZMMMA

Monumental mason's website

headstones.org.nz

Fully custom developed Website for the New Zealand Master Monumental Masons Association.

With an ambitious design which includes horizontal scrolling; this site includes information about the association, a gallery, news section, member details and a contact form — managed through a custom developed CMS backend. The CMS system also provides a members' portal for file sharing.

Developed while employed at T&D with visual design by Kristina Marotzke.
